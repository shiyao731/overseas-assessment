# 天眼查站点经验指南

## 站点概述
- **域名**: www.tianyancha.com
- **类型**: 企业信息查询平台
- **核心价值**: 企业工商信息、融资历史、股权结构、法律诉讼

## 推荐访问方式

### 方式一：Jina Reader (推荐)
```
https://jina.ai/http://www.tianyancha.com/search?key={企业名称}
```
- **优势**: Token 效率高，无需处理反爬虫
- **适用**: 快速获取企业基本信息

### 方式二：浏览器自动化 (备用)
当 Jina Reader 无法获取完整信息时使用：
- 使用 Playwright 工具访问
- 处理可能的验证码
- 提取详细企业数据

## 关键信息提取点

### 企业基本信息
- 企业全称
- 成立日期
- 注册资本
- 法人代表
- 经营状态

### 财务与融资
- 融资轮次与金额
- 投资机构
- 估值信息
- 营收规模（如有披露）

### 股权结构
- 股东构成
- 持股比例
- 对外投资

### 风险信息
- 法律诉讼
- 经营异常
- 失信信息

## 搜索技巧
1. 使用企业全称搜索最准确
2. 缩写可能返回多个结果，需要筛选
3. 关注"融资历史"标签页
4. 查看"股东信息"了解股权结构
