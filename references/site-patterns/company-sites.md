# 公司官网通用访问指南

## 站点类型
- 上市公司官网
- 非上市公司官网
- 集团公司/控股公司官网

## 推荐访问方式

### 方式一：Jina Reader (首选)
```
https://jina.ai/http://{company-website-url}
```
- **优势**:
  - 处理动态加载内容
  - Token 效率高（通常 < 2000 tokens）
  - 提取纯文本，无广告干扰

### 方式二：WebFetch
适用于静态页面或 Jina 不可用时

### 方式三：浏览器自动化
当需要：
- 查看交互式内容（视频、图表）
- 处理复杂的单页应用
- 提取渲染后的数据

## 关键页面优先级

### 优先级 1：投资者关系 (IR)
- 路径通常：`/investor` 或 `/ir`
- 关键信息：
  - 财报（年度/季度）
  - 投资者演示材料
  - 业绩公告
  - 股票信息（如上市）

### 优先级 2：关于我们
- 路径通常：`/about` 或 `/company`
- 关键信息：
  - 公司简介
  - 发展历程
  - 管理团队
  - 企业文化

### 优先级 3：新闻中心
- 路径通常：`/news` 或 `/media`
- 关键信息：
  - 最新动态
  - 产品发布
  - 战略合作
  - 海外业务进展

### 优先级 4：业务/产品
- 路径通常：`/business` 或 `/products`
- 关键信息：
  - 核心产品与服务
  - 业务板块
  - 技术能力
  - 客户案例

## 内容提取技巧

### 上市公司特有信息
- 股票代码与交易所
- 市值（实时）
- 财务数据：
  - 营业收入
  - 净利润
  - 资产负债表
  - 现金流量表

### 海外业务信息
- 海外分公司/子公司列表
- 国际业务占比
- 海外投资项目
- 出口数据

### 技术实力指标
- 研发投入占比
- 专利数量
- 研发团队规模
- 技术奖项/认证

## URL 推测技巧
1. `www.{company-name}.com`
2. `www.{company-name}.cn` (中国企业)
3. `{company-name}.group` (集团公司)
4. 使用 WebSearch 搜索 "{公司名} 官网" 验证
