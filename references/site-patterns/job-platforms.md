# 招聘平台站点经验指南

## 涵盖平台
- 猎聘 (www.liepin.com)
- 拉勾 (www.lagou.com)
- BOSS直聘 (www.zhipin.com)
- 智联招聘 (www.zhaopin.com)

## 核心价值
评估企业出海能力的"人才维度"指标：
- 海外招聘岗位 → 国际化程度
- 招聘规模 → 业务扩张情况
- 薪资水平 → 企业实力
- 岗位要求 → 技术壁垒

## 推荐访问方式

### 方式一：WebSearch (推荐，最稳定)
```
搜索关键词：
- "{公司名} 海外招聘 猎聘"
- "{公司名} 国际化岗位 招聘"
- "{公司名} 海外人才"
```
- **优势**: 避开反爬虫，获取搜索结果摘要
- **适用**: 快速了解是否有海外招聘

### 方式二：Jina Reader (谨慎使用)
```
https://jina.ai/http://www.liepin.com/company/{company-id}/
```
- **限制**: 可能被反爬虫拦截
- **适用**: 知道具体企业 ID 时

### 方式三：浏览器自动化 (最后手段)
- 使用 Playwright 工具
- 需要处理：
  - 滑动验证码
  - 登录要求
  - 频繁访问限制

## 关键信息提取

### 海外岗位指标
- 岗位名称包含"海外"、"国际"、"跨境"
- 工作地点在海外国家/城市
- 要求外语能力（英语、日语等）
- 驻外/出差要求

### 国际化程度评估
| 海外岗位数量 | 国际化程度评估 |
|------------|--------------|
| 0 | 🔴 无明显国际化动作 |
| 1-5 | 🟡 初步探索海外 |
| 5-20 | 🟢 有国际化布局 |
| 20+ | 🌍 深度国际化 |

### 岗位类型分析
- **海外销售/商务**: 市场开拓型
- **海外研发/技术**: 技术输出型
- **海外运营**: 本地化运营型
- **外派人员**: 业务实体型

## 搜索技巧
1. **组合搜索**: "公司名 + 海外 + 招聘"
2. **时间过滤**: 关注近 6 个月的招聘动态
3. **岗位分析**: 区分"真海外岗位"vs"国内涉外岗位"
4. **跨平台验证**: 同一企业在多个平台都有海外招聘 → 可信度高
