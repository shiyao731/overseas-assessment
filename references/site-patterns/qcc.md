# 企查查站点经验指南

## 站点概述
- **域名**: www.qcc.com
- **类型**: 企业信息查询平台
- **核心价值**: 类似天眼查，提供企业工商、融资、知识产权信息

## 推荐访问方式

### 方式一：Jina Reader (推荐)
```
https://jina.ai/http://www.qcc.com/web/search?key={企业名称}
```
- **优势**: Token 效率高，内容结构化
- **适用**: 快速获取企业基本信息

### 方式二：浏览器自动化 (备用)
当需要获取详细信息或遇到访问限制时：
- 使用 Playwright 工具
- 处理登录/验证要求
- 提取完整的企业档案

## 关键信息提取点

### 企业基本信息
- 工商注册信息
- 统一社会信用代码
- 企业类型
- 成立日期与注册资本

### 知识产权
- 专利信息
- 商标注册
- 软件著作权
- 网站备案

### 融资与投资
- 融资历史
- 投资事件
- 并购记录

## 搜索技巧
1. 优先使用企业完整注册名称
2. 查看知识产权了解技术实力
3. 融资历史可以验证企业发展阶段
4. 关注对外投资了解业务布局
