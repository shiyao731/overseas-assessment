## 核心升级

### 1. 对象公司+母公司双轨打分机制

**问题背景**：
- 用户反馈：评估对象公司时，母公司信息丰富但对象公司数据不足
- 需求：希望充分利用母公司的数据和资源，挖掘集团层面的合作机会

**v3.2.0 解决方案**：

#### 1.1 企业结构自动识别
- **新增步骤0**：在每次评估开始时，首先搜索母公司信息
- **搜索关键词**："{公司名} 母公司/股权结构/实控人"
- **数据源**：企查查、天眼查、公司官网、年报

#### 1.2 双轨独立评估
当识别到母公司时，自动启用双轨评估流程：
- **母公司独立评估**：对母公司执行完整的12维度评分
- **对象公司评估策略**：
  - 信息充分（>70%）→ 独立评估
  - 信息部分（30-70%）→ 混合评估
  - 信息不足（<30%）→ 基于业务特性推断

#### 1.3 集团机会挖掘
自动识别并列出：
- 集团内其他出海板块
- 交叉销售机会
- 资源共享潜力
- 战略切入点

#### 1.4 交叉销售线索生成
输出格式：
```markdown
| 业务单元 | 出海状态 | 推荐服务 | 协同效应 | 联系优先级 |
```

### 2. 多层级动态权重校准机制

**问题背景**：
- 用户反馈：原有的4类出海阶段权重调整过于粗略
- 需求：希望根据企业"出海阶段+体量+行业"三重特征，自动应用精细化的权重配置

**v3.2.0 解决方案**：

#### 2.1 四类出海企业详细画像

| 企业类型 | 核心痛点 | 权重提升维度 | 典型话术 |
|---------|---------|-------------|---------|
| **初次出海** | 不知道从哪里开始 | 监管准入(+5%)、市场(+5%) | "帮您快速了解目标市场准入要求" |
| **已有项目** | 运营效率不够高 | 运营效率(+5%)、供应链(+3%) | "帮您提升现有工厂的运营效益" |
| **大型集团** | 需要统筹全球布局 | 战略协同(+18%)、合规风控(+20%) | "帮您规划集团层面的出海战略" |
| **中小创新** | 预算有限需快速验证 | 成本控制(+18%)、增长(+18%) | "帮您低成本快速进入目标市场" |

#### 2.2 详细权重调整矩阵

**初次出海企业**：
- 市场规模与增长：13% → 18% （+5%）
- 监管与准入：20% → 25% （+5%）
- 运营效率：15% → 10% （-5%）
- 核心话术："帮您快速了解目标市场，规避准入风险"

**已有海外项目企业**：
- 市场规模与增长：13% → 10% （-3%）
- 运营效率：15% → 20% （+5%）
- 本地化团队：新增 → 12% （+2%）
- 核心话术："帮您提升海外运营效率和本地化水平"

**大型集团企业**：
- 战略协同性：新增 → 18%
- 品牌影响力：新增 → 15%
- 合规风控：合并到监管维度
- 核心话术："帮您规划集团全球布局和风险控制"

**中小型创新企业**：
- 市场规模与增长：13% → 18% （+5%）
- IP与技术：10% → 14% （+4%）
- 核心话术："帮您低成本快速验证海外市场"

#### 2.3 权重调整决策树

```
Step 1: 判断出海阶段
├─ 无海外业务 → [初次出海企业]
├─ 海外收入 < 30% → [已有海外项目企业]
├─ 营收 > 50亿 且 多业务板块 → [大型集团企业]
└─ 营收 < 5亿 且 技术驱动 → [中小型创新企业]

Step 2: 应用对应权重配置
Step 3: 行业微调（可选）
```

## 使用建议

### 对于评估人员
1. **首次使用**：仔细阅读"示例对话流程"中的示例2，了解双轨评估流程
2. **权重配置**：根据企业出海阶段，参考"详细权重调整矩阵"选择合适的话术方向
3. **集团机会**：当识别到母公司时，重点关注"集团机会挖掘"输出

### 对于销售人员
1. **话术方向**：根据企业类型，使用对应的"典型话术"
2. **交叉销售**：利用"交叉销售线索生成"输出，寻找多点触达机会
3. **痛点识别**：根据"四类出海企业详细画像"，快速定位客户核心痛点
